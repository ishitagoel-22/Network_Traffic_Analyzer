# Network Traffic Analyzer

This project analyzes real network traffic using Wireshark and Java. It extracts insights like protocol distribution (TCP/UDP) and identifies the most active IP addresses.

---

##  Features
- Capture network packets using Wireshark
- Analyze TCP and UDP traffic
- Calculate percentage distribution
- Identify top IP addresses
- Visualize data using pie chart (JFreeChart)

---

## 🛠 Tech Stack
- Java
- Wireshark
- JFreeChart
- Maven

---

##  Project Workflow
1. Capture packets using Wireshark
2. Export data as CSV
3. Process data using Java
4. Generate insights and visualization

---

##  Output Example
- Total packets
- TCP / UDP count
- Percentage distribution
- Top IP addresses
- Pie chart visualization

---

##  Insight
Most network traffic is TCP, indicating web-based communication such as HTTP/HTTPS.
